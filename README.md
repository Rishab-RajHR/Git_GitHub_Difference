Git is a distributed version control system used to track changes in source code.


GitHub is a cloud-based hosting platform for Git repositories.


Git works locally on your machine; GitHub works online.


Git manages commits, branches, merges, and history.


GitHub provides collaboration tools like pull requests, issues, and project boards.


Git does not require the internet; GitHub needs an internet connection.


Git is a command-line tool; GitHub offers a web interface and additional features.


GitHub enhances Git with team collaboration, backups, and CI/CD integrations.
